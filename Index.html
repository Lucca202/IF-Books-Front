Bem-vindo ao IF-Books

Este projeto foi desenvolvido pelo aluno Lucca Bonato Costa do 3º semestre do curso de Introdução ao Desenvolvimento Web do Instituto Federal de São João da Boa Vista, como parte da disciplina IDSW.

O sistema permite gerenciar livros e autores de forma simples e moderna.

Ver Livros Ver Autores

Sobre o Projeto

O IF-Books simula um sistema de biblioteca acadêmica, utilizando tecnologias como HTML, CSS, JS, Bootstrap, Spring Boot e MySQL, integrando o front-end ao back-end via API REST.

Um ambiente de aprendizado prático em desenvolvimento web moderno.
